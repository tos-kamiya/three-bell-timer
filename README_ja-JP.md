# three-bell-timer

軽量かつ透過性のあるプレゼンテーション用タイマーです。デスクトップ上にリアルタイムで進行する水平プログレスバーを表示します。

## 特長

### 3鈴の時間管理

- **1鈴**: 終了が近いことを通知
- **2鈴**: プレゼンテーションの終了時刻を示す
- **3鈴**: 質疑応答終了時刻を示す

### プログレスバー

- 分ごとに角丸マーブルを表示
  - **経過前**: 薄い色
  - **経過後**: 濃い色
- **100msごと**に更新し、スムーズにバーが進行

### 一時停止時の表示

- 一時停止状態では左端に大きな **▶** アイコン
- 1鈴・2鈴・3鈴の数値マーカーをバー内に表示
- ▶ をクリックすると即座に開始／再開

### ディスプレイ & 配置制御

- **デフォルト**: 全ディスプレイに表示
- `--display all` / `--display 0` / `--display 0,1,2` で特定画面を指定
- `--pos top` / `--pos bottom` で上下配置を指定
- メニューの **Cycle Display Target** で「全画面 → 0番 → 1番 → …」と切り替え可能

### システムトレイアイコン

- タスクバーには表示せず、システムトレイにアイコンを配置
- トレイメニューで以下の操作が可能:
  - 開始／一時停止
  - 鈴時刻の変更
  - ディスプレイ切り替え
  - 配置変更（上端／下端）
  - アプリケーション終了

## インストール

pipx によりインストールすることを推奨します。

```bash
pipx install git+https://github.com/tos-kamiya/three-bell-timer
```

pipx 未導入の場合は、

```bash
python3 -m pip install --user pipx
python3 -m pipx ensurepath
```

あるいは、

```bash
git clone https://github.com/tos-kamiya/three-bell-timer
pip install .
```

## 使い方

```bash
# デフォルト設定: 1鈴=10分、2鈴=15分、3鈴=20分、全ディスプレイ、上端、バーの高さ10px
3bt

# 1つの値: 全鈴を12分に設定
3bt 12

# 2つの値: 1鈴=10分、2鈴・3鈴=20分
3bt 10 20

# 3つの値: 1鈴=10分、2鈴=15分、3鈴=20分、ディスプレイ1、下端、バーの高さ12px
3bt 10 15 20 --display 1 --pos bottom --pixel-height 12

# 複数ディスプレイ: 0番と2番のみ表示
3bt --display 0,2

# 全ディスプレイ明示
3bt --display all
```
